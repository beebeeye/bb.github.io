Description

The outcome of this task a rough master plan.

Why Pain?

This task is restricted by many standards, like those for and setback, etc. Sometimes satisfying one set of codes may violate another set of codes. This is where the challenges are.

Workarounds

The architects would like the codes of architecture design to be integrated into the CAD software. The architects would like to use the feature in the following way: (1) When starting a new project, the architect picks the set of codes to apply. During the design, if any part of the design violates a code, a warning box is popped up. (2) The architects have their drawings inspected against the codes by the software before submission to the regulatory agencies. (3) Design software can automatically lay out all the feasible designs under preset codes and rules for architects to select.
